Manual de referência da biblioteca color.h 1.0.3

Funções

coutc("text", style::FOREGRAUND)
FOREGRAUND deve ser substituído por uma das possibilidades de FOREGROUND
Escreve um texto em determinada cor

coutc("text", style::BACKGROUND)
BACKGROUND deve ser substituído por uma das possibilidades de BACKGROUND
Escreve um texto com determinada cor de fundo

coutc("text", style::STYLE)
STYLE deve ser substituído por uma das possibilidades de STYLE
Escreve um texto com determinado estilo

coutc("text", style::FOREGROUND, style::BACKGROUND)
FOREGRAUND deve ser substituído por uma das possibilidades de FOREGROUND e BACKGROUND deve ser substituído por uma das possibilidades de BACKGROUND
Escreve um texto em determinada cor e com determinada cor de fundo

coutc("test", style::FOREGROUND, style::STYLE)
FOREGRAUND deve ser substituído por uma das possibilidades de FOREGROUND e STYLE deve ser substituído por uma das possibilidades de STYLE
Escreve um texto em determinada cor e com determinada cor de fundo

coutc("text", style::BACKGROUND, style::STYLE)
BACKGROUND deve ser substituído por uma das possibilidades de BACKGROUND e STYLEdeve ser substituído por uma das possibilidades de STYLE
Escreve um texto com determinada cor de fundo e com determinado estilo

coutc("text", style::FOREGROUND, style::BACKGROUND, style::STYLE)
FOREGUARD deve ser substituído por uma das possibilidades de FOREGROUND, BACKGROUND deve ser substituído por uma das possibilidades de BACKGROUND e STYLE deve ser substituído por uma das possibilidades de STYLE
Escreve um texto em determinada cor e com determinada cor de fundo e em determinado estilo

Possibilidades de estilos

Aqui estão as possibilidades de FOREGROUND:
        fBLACK = letra preta
        fRED = letra vermelha
        fGREEN = letra verde
        fYELLOW = letra amarela
        fBLUE = letra azul
        fMAGENTA = letra magenta
        fCYAN = letra ciano
        fWHITE = letra branca
        fBLACKs = letra preta estrondosa
        fREDs = letra vermelha estrondosa
        fGREENs = letra verde estrondosa
        fYELLOWs = letra amarela estrondosa
        fBLUEs = letra azul estrondosa
        fMAGENTAs = letra magenta estrondosa
        fCYANs = letra ciana estrondosa
        fWHITEs = letra branca estrondosa

Aqui estão as possibilidades de BACKGROUND:
        bBLACK = fundo preto
        bRED = fundo vermelho
        bGREEN = fundo verde
        bYELLOW = fundo amarelo
        bBLUE = fundo azul
        bMAGENTA = fundo magenta
        bCYAN = fundo ciano
        bWHITE = fundo branco
        bBLACKs = fundo preto estrondoso
        bREDs = fundo vermelho estrondoso
        bGREENs = fundo verde estrondoso
        bYELLOWs = fundo amarelo estrondoso
        bBLUEs = fundo azul estrondoso
        bMAGENTAs = fundo magenta estrondoso
        bCYANs = fundo ciano estrondoso
        bWHITEs = fundo branco estrondoso

Aqui estão as possibilidades de STYLE:
        sRESET = estilo de letra padrão
        sBOLD = cor da letra fica mais forte
        sUNDERLINE = sublinhado
        sINVERSE = inverte a cor do fundo com a cor da letra
